# shoppingcart

Gekozen framework is Vue.js, omdat ik daar de laatste periode het meest mee heb gewerkt en daarmee het meest bekend ben met het gebruik van state management.
Voor het state management heb ik daarbij Vuex gebruikt, deze wordt het meest gebruikt binnen Vue wanneer je state management wil gebruiken. Voor de implementatie 
hiervan heb ik gekozen om dit snel in de main.js neer te zetten, om daarmee tijd te besparen.

In de state management heb ik 2 actions toegevoegd voor toevoegen van cart items en favorieten. Dit dient te worden uitgebreid met mogelijkheden om ook cart items te verwijderen of een aantal van een
cart item te verlagen of op te hogen.

Voor de styling maak ik gebruik van scss om makkelijker style regels in elkaar te zetten en daarmee deze regels in te korten. Ik heb de responiveness om het op mobiel te gebruiken ingesteld op 
iPhone 6/7/8 met een resolutie van 375 x 667, wanneer je deze applicatie verder zou uitbreiden moeten hier uiteraard meer regels of meer generiekere regels voor worden toegepast.


Tevens heb ik FontAwesome gebruikt voor icons, deze library gebruik ik vaker omdat deze zeer uitgebreid is en gemakkelijk in gebruik is.
